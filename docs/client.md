# Клиентские модули

Клиент — нативные ES-модули в браузере (без сборщика), лежат в `public/js/`.
Точка входа подключается из `index.html` как `<script type="module">`.
Принципы — в [architecture.md](architecture.md).

## Точка входа и конфигурация

| Модуль | Назначение |
|---|---|
| `js/main.js` | Оркестратор: `render()`, привязка событий, сценарии входа/выхода, `init()`. При входе в комнату (`JOINED`) сразу прописывает `?room=КОД` в адресной строке, чтобы ссылку-приглашение можно было скопировать прямо из URL. Модалки (настройки, ник, выход) закрываются кликом по затемнённому фону. Если вход в комнату не удался (`ERROR` — например, сервер перезапущен и комнаты больше нет), клиент сбрасывает сохранённый код, останавливает авто-переподключение и возвращает игрока на главный экран. Текст ошибки показывается только при явной попытке входа (ввод кода или ссылка-приглашение); при тихом восстановлении сессии по сохранённому коду ошибка не показывается. |
| `js/config.js` | `DEFAULTS` (настройки по умолчанию) и `WORDS` (словарь без дубликатов). |

## Интерфейс (`js/ui/`)

| Модуль | Назначение |
|---|---|
| `ui/screens.js` | `show` — навигация между экранами. |
| `ui/board.view.js` | `renderBoard` — игровое поле (CSS Grid N×N с равными колонками `1fr` и рядами `grid-auto-rows:1fr`, центрировано, ограничено 910×600; все карты одинакового размера независимо от длины слова — раньше flex-перенос делал ряды разной высоты). Угадывающему невскрытые карты показываются нейтральным светлым фоном — цвет ему неизвестен до открытия. Клик по карте — **голос** агента (task 1): кружки проголосовавших (цвет по id, свой — в белой обводке) рисуются в углу карты, а при единогласии снизу появляется 2-сек лоадер перед открытием. При старте партии капитану его карты ненадолго подсвечиваются (`spotlight`). Капитану уже открытые карты приглушаются прозрачностью (`dim-guessed`, task 3, `opacity:0.2`) — цвет угаданной карты остаётся виден сквозь приглушение, чтобы внимание было на оставшихся. |
| `ui/teams.view.js` | `renderTeams` — списки команд (кружок-аватар стабильного цвета по id игрока слева, пометка «это вы» справа), счёт (скрыт в лобби), карточка «Наблюдатели:» в шапке и ссылки-действия «Стать капитаном» / «Войти в команду» (видны в лобби и на паузе; «Стать капитаном» скрыта, когда в команде уже есть капитан — один капитан на команду). Карточка команды, чей сейчас ход, получает **неоновую подсветку** (`active-turn`, task 4). Плюс подвал карточки: **ввод подсказки** капитаном текущей команды (одно поле «слово + цифра», отправка по Enter или кнопке «+», валидация «нельзя два слова»), **история подсказок** команды (слово + число рядом) и кнопка **«Пропустить ход»** как голосование (task 1): кружки проголосовавших за пропуск под кнопкой и 2-сек лоадер при единогласии; клик шлёт `endTurn` (голос, повторный — снимает). `bindTeamActions` навешивает обработчики один раз (DOM подвала статичен). |
| `ui/status.view.js` | `renderStatus` — верхняя строка фазы (видна только в лобби/после игры) и нижняя фиксированная карточка таймера `#timer-bar` (только в активной игре, приклеена к нижнему краю по центру): подпись «Лидер думает:» (фаза clue) / «Команда думает:» (фаза guess) + таймер + одна кнопка паузы-переключателя (PAUSE/RESUME, подпись «⏸ Пауза» ↔ «▶️ Возобновить», стиль `btn-ghost` единый для обоих состояний). На паузе подпись и значение таймера не меняются — слева остаётся обычный текст «думает», справа показывается оставшееся время (без слова «Пауза» и иконки `⏸`). Чей ход в активных фазах показывает неоновая подсветка карточки команды (task 4). |
| `ui/controls.view.js` | `renderControls` — кнопки старт/новая игра/в лобби/перемешать. |
| `ui/settings.view.js` | `renderSettings`, `readSettingsForm` — форма настроек; кнопка «Перезапустить игру» (только хост, шлёт `newGame`). |
| `ui/log.view.js` | `renderLog` — журнал событий. |
| `ui/modals.js` | `renderWin` — при завершении игры заливает экран цветом команды-победителя: круг расходится из центра на полэкрана и в конце гаснет (task 2), отдельной модалки победы нет. `copyFeedback`, `openNickModal`. |
| `ui/toast.js` | `showToast` — всплывающие уведомления. |
