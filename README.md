# 🕵️ Codenames — онлайн

Браузерная версия игры **Codenames** с онлайн-комнатами на Node.js + WebSocket.
Две команды — 🔴 красные и 🔵 синие — соревнуются, кто первым откроет все свои
карты. Капитан команды видит цвета карт и даёт подсказку «слово + число», а
угадывающие по ней открывают карты. Открыть карту-убийцу — мгновенное поражение.

## Возможности

- 🎲 Поле **5×5** или **6×6** (выбирается в настройках)
- 📚 Базовый словарь из ~1000 простых русских слов
- ⚙️ Настройки таймеров:
  - **Время первого хода** — длительность фазы подсказки капитана;
  - **Время ответа** — длительность фазы угадывания команды;
  - **Доп. время** — секунды, добавляемые за каждый верный ответ.
- ⏸ **Пауза / возобновление** игры
- 🌐 **Игра по сети** с друзьями через комнаты (код из 4 символов)
- 👤 Авторизация по нику, **смена ника** в любой момент
- 🌑 Тёмная тема
- 🔀 **Перемешивание команд** одной кнопкой
- 👀 **Наблюдатели**: игроки заходят без команды и сами выбирают сторону
- 🔁 Авто-переподключение при разрыве связи

## Запуск сервера

```bash
npm install
npm start
```

Сервер поднимется на **http://localhost:3000**. Порт можно переопределить
переменной окружения `PORT`:

```bash
PORT=8080 npm start
```

## Как подключиться другому игроку

1. Создайте комнату — вы получите **код из 4 символов** (в шапке «Комната») и
   кнопку «🔗» со ссылкой-приглашением.
2. Дайте другу адрес вашего сервера, подставив код комнаты в `?room=КОД`:
   - **в той же Wi-Fi/сети** — `http://<ваш-IP>:3000/?room=КОД`
     (нужно открыть порт `3000` в брандмауэре);
   - **через интернет** — поднимите туннель и дайте публичный `https://…/?room=КОД`.
     Проще всего, без установки (нужен лишь штатный `ssh`):

     ```bash
     npm run tunnel              # печатает домен и кликабельную ссылку
     npm run tunnel -- -Room КОД # сразу готовая ссылка с кодом комнаты
     ```

     Команда поднимает туннель localhost.run и выводит готовый адрес. Окно не
     закрывать, пока идёт игра. Альтернатива — Cloudflare Tunnel (`cloudflared`).
3. Друг открывает ссылку, вводит ник — и сразу попадает в вашу комнату. (Можно
   передать только **код** — он вставит его в поле «КОД» → «Войти».)

Подробные пошаговые инструкции (IP, правило брандмауэра, туннели localhost.run и
Cloudflare) — в [docs/hosting.md](docs/hosting.md).

## Как играть

1. Введите никнейм → создайте комнату или войдите по коду.
2. Новые игроки заходят как **наблюдатели** (без команды) — они видны в шапке
   «Наблюдатели:». Каждый сам выбирает сторону: «Стать капитаном» или «Войти в
   команду» (**🔴 красные / 🔵 синие**). Хост может разом раскидать всех кнопкой
   «🔀 Перемешать команды».
3. **Хост** (👑) настраивает поле и таймеры и жмёт «▶️ Начать игру».
4. Капитан текущей команды видит цвета карт и даёт подсказку: **слово + число**.
5. Угадывающие команды открывают карты:
   - своя карта → можно продолжать (за верный ответ +доп. время);
   - чужая / нейтральная → ход переходит;
   - 💀 **карта-убийца** → мгновенное поражение.
6. Побеждает команда, первой открывшая все свои карты.
